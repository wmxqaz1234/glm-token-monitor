# API 配置提示气泡优化 - 侧边悬浮卡片方案

## ✨ 优化内容

将原本紧贴在宠物面板内的气泡改为**侧边悬浮卡片**，彻底解决空间不足和文字被遮挡的问题。

---

## 🎨 新的设计

### 视觉效果

**侧边悬浮卡片**：
- 位置：显示在宠物面板右侧（外部）
- 尺寸：160-200px 宽度，高度自适应
- 动画：从右侧滑入 + 上下浮动
- 样式：渐变背景 + 毛玻璃效果 + 发光阴影

**卡片内容**：
```
┌─────────────────────┐
│ 🔑                  │ ← 大图标（带摇晃动画）
│                    │
│ 需要配置 API Key   │ ← 标题（加粗，蓝色发光）
│                    │
│ 请先配置 API Key   │ ← 描述文字
│ 才能查看使用量     │
│                    │
│      [→ 去设置]    │ ← 操作按钮（带图标）
└─────────────────────┘
```

### 交互优化

1. **显示时机**：启动 2 秒后自动显示
2. **自动隐藏**：10 秒后自动消失
3. **点击隐藏**：用户点击任何地方后立即隐藏
4. **点击卡片**：直接打开设置面板

---

## 📐 技术实现

### 1. 位置定位

```css
.api-config-bubble {
  left: 100%;                    /* 从容器右边缘开始 */
  top: 50%;                     /* 垂直居中 */
  transform: 
    translateY(-50%)             /* 垂直居中 */
    translateX(8px);            /* 向右偏移 8px */
  z-index: 1000;                /* 确保在最上层 */
}
```

### 2. 容器溢出处理

```css
.pet-widget {
  /* 关键修改：允许内容溢出 */
  overflow: visible;            /* 从 hidden 改为 visible */
}
```

### 3. 箭头指向

```css
.api-config-bubble::after {
  /* 左侧箭头指向宠物 */
  left: -8px;
  top: 50%;
  transform: translateY(-50%);
  border-right: 8px solid #3b82f6;  /* 箭头向右 */
  border-top: 6px solid transparent;
  border-bottom: 6px solid transparent;
}
```

### 4. 动画效果

```css
/* 入场动画：从右侧滑入 */
@keyframes slideInRight {
  from {
    opacity: 0;
    transform: translateY(-50%) translateX(-20px) scale(0.9);
  }
  to {
    opacity: 1;
    transform: translateY(-50%) translateX(8px) scale(1);
  }
}

/* 持续动画：上下浮动 */
@keyframes api-float {
  0%, 100% {
    transform: translateY(-50%) translateX(8px);
  }
  50% {
    transform: translateY(-52%) translateX(8px);
  }
}
```

## 🎯 优化对比

### 优化前

| 问题 | 描述 |
|------|------|
| ❌ 空间受限 | 120x120 容器内显示 |
| ❌ 文字被截断 | 只能显示简短文字 |
| ❌ 视觉压抑 | 紧贴宠物，显得拥挤 |
| ❌ 不够醒目 | 小气泡容易被忽略 |

### 优化后

| 优势 | 描述 |
|------|------|
| ✅ 空间充足 | 显示在容器外部，无尺寸限制 |
| ✅ 完整内容 | 可显示标题、描述、按钮 |
| ✅ 视觉舒适 | 侧边悬浮，不遮挡宠物 |
| ✅ 醒目提示 | 大卡片 + 动画，无法忽视 |

## ⚙️ 智能隐藏逻辑

### 自动隐藏触发条件

1. **时间触发**：显示 10 秒后自动隐藏
2. **交互触发**：用户点击页面任何地方后隐藏
3. **配置完成**：检测到 API Key 后隐藏

### 代码实现

```typescript
// 显示气泡
showApiConfigBubble.value = true

// 10秒后自动隐藏
setTimeout(() => {
  showApiConfigBubble.value = false
}, 10000)

// 用户点击后隐藏
const hideBubble = () => {
  showApiConfigBubble.value = false
}
setTimeout(() => {
  document.addEventListener('mousedown', hideBubble, { once: true })
}, 100)
```

## 🔍 适配性

### 不同屏幕尺寸

**小屏幕** (≤ 1366x768)：
- 卡片宽度：160px
- 字体大小：适当缩小

**大屏幕** (≥ 1920x1080)：
- 卡片宽度：200px
- 字体大小：正常显示

### 多显示器

- 卡片始终跟随宠物面板
- 在主显示器上显示
- 不会被屏幕边界裁剪

## 📱 其他可选方案

### 方案对比

| 方案 | 优势 | 劣势 |
|------|------|------|
| **侧边悬浮卡片** ✅ | 空间充足、视觉舒适 | 需要修改容器 overflow |
| 顶部横幅通知 | 醒目、不遮挡 | 遮挡顶部空间 |
| Tooltip | 简洁、不占空间 | 需要悬停才能看到 |
| 独立窗口 | 完全独立 | 需要管理窗口生命周期 |

3. **模态对话框**：首次启动强制配置
4. **Toast 通知**：轻量级提示
