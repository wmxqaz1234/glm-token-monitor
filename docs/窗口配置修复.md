# 窗口配置修复 - 侧边悬浮卡片动画问题

## ❌ 问题原因

### 原始配置问题

```json
{
  "width": 96,
  "height": 96,
  "minWidth": 96,
  "minHeight": 96,
  "maxWidth": 96,        // ❌ 限制了最大宽度
  "maxHeight": 96,       // ❌ 限制了最大高度
}
```

**问题分析**：
1. 窗口被锁定在 96x96
2. PetWidget 容器是 120x120，已经超出窗口
3. 气泡要显示在窗口外部，但窗口无法扩展
4. 导致气泡被截断，动画无法正常显示

---

## ✅ 解决方案

### 1. 调整窗口大小

```json
{
  "width": 120,           // ✅ 改为 120，匹配容器大小
  "height": 120,          // ✅ 改为 120
  "minWidth": 96,         // ✅ 保持最小尺寸
  "minHeight": 96,
  "maxWidth": 400,       // ✅ 移除限制，允许扩展
  "maxHeight": 400,      // ✅ 移除限制
}
```

**效果**：
- 窗口默认 120x120，完整显示宠物
- 可以调整到最大 400x400（如果用户需要）
- 气泡可以显示在窗口外部

### 2. 优化气泡定位

```css
.api-config-bubble {
  position: absolute;     /* ✅ 确保正确定位 */
  left: 100%;             /* 从容器右边缘开始 */
  transform:
    translateY(-50%)     /* 垂直居中 */
    translateX(12px);   /* 向右偏移 12px（增加间距）*/
  pointer-events: auto;   /* ✅ 确保可点击 */
}
```

### 3. 修正动画

```css
@keyframes slideInRight {
  from {
    opacity: 0;
    transform: translateY(-50%) translateX(0px) scale(0.9);
  }
  to {
    opacity: 1;
    transform: translateY(-50%) translateX(12px) scale(1);
  }
}

@keyframes api-float {
  0%, 100% {
    transform: translateY(-50%) translateX(12px);
  }
  50% {
    transform: translateY(calc(-50% - 4px)) translateX(12px);
  }
}
```

## 💡 其他优化建议

### 窗口大小选择

**当前设置**：120x120

**考虑因素**：
- 宠物本身占用约 80x80
- 加上 padding 和光晕，120x120 比较合适
- 为未来可能的 UI 扩展预留空间

**可选方案**：
- 更小：96x96（节省空间）
- 更大：150x150（更多空间）

### 用户可调整

由于移除了 maxWidth/maxHeight 限制，用户可以：
1. 通过调整窗口大小来查看更多内容
2. 在需要时缩小窗口节省空间
3. 未来可以添加"紧凑模式"选项
